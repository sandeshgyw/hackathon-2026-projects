# CareDevi AI Innovation Hackathon 2026

## Project Context

This is a hackathon repo for the CareDevi AI Innovation Hackathon 2026 (Apr 25 – Apr 26, CDT).
Focus: AI-driven solutions for real-world healthcare challenges.

**Hard deadline: April 26, 2026 at 4:00 PM CDT. No extensions.**

## Repository Structure

```
projects/
└── <team-name>/
    ├── README.md          # Problem, approach, architecture, setup, team credits
    ├── responsible-ai.md  # Data sources, model choices, bias, failure cases
    ├── src/               # Source code
    └── demo/              # Demo video or screenshots
```

## Guidelines

- Commit early and often — commit history is scored (5% of total)
- Scope tightly: a polished small project beats a broken ambitious one
- All four deliverables are mandatory: GitHub repo, README, 3-min demo, Responsible AI doc
- Required milestone commits: Saturday Apr 25, 4:00–6:00 PM CDT

## Scoring Weights

- Real-World Impact: 25%
- Technical Innovation: 15%
- User Experience: 15%
- Presentation: 10%
- Feasibility: 5%
- Mentor feedback (check-ins, commits, teamwork, responsible AI): 20%

## Coding Preferences

- Keep code simple and shippable — avoid over-engineering
- Document as you go (inline comments for non-obvious logic)
- Prefer working features over perfect architecture
