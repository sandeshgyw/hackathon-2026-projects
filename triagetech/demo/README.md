## ❗ Problem Statement
In many regions, patients often face delays in receiving medical attention due to limited healthcare resources and difficulty in identifying the urgency of their condition. This leads to overcrowded hospitals and delayed treatment for critical patients.

---

## 💡 Solution Description
TriageTech is an AI-assisted medical triage system that helps users:
- Analyze symptoms and classify urgency (Emergency / Moderate / Self-care)  
- Provide instant guidance on next steps  
- Suggest nearby clinics or hospitals  
- Support both English and Nepali language  

---

## 🛠 Tech Stack
- **Frontend:** React.js / HTML / CSS  
- **Backend:** FastAPI (Python)  
- **AI Logic:** Python-based symptom classification  
- **Tools:** Git, GitHub, Uvicorn
